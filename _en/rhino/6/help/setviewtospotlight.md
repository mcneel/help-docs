Set View to Spotlight
The SetViewToSpotlight command matches the view to a spotlight direction.
This command changes the current view so that the camera position matches the spotlight source location and the camera target matches the light target point (center of base of the cone). The camera angle matches the light cone angle. This can verify a light's view as an analysis aid in setting up a scene.
